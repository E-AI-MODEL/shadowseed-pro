# Latest test artifacts

Deze map is opgebouwd uit de meest recente geslaagde standaardrun `Checks en benchmark-resultaten`.

De map bevat een platte kopie van de belangrijkste JSON-, Markdown- en SVG-bestanden uit de test-artifacts. Gebruik `manifest.json` om te zien uit welk artifact elk bestand komt.

Bij dubbele bestandsnamen krijgt het latere bestand een prefix met de artifactnaam. De originele artifactstructuur blijft behouden in `results/artifacts/`.
