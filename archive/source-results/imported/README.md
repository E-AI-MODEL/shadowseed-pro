# Imported result bundles

Deze map is bedoeld voor handmatig aangeleverde of achteraf geïmporteerde resultaatbundels.

Belangrijke regel:

- `results/latest/` blijft gereserveerd voor de canonieke snapshot van de standaard publicatieroute.
- geïmporteerde bundels krijgen daarom een eigen submap met duidelijke provenance.

Gebruik deze map voor review, vergelijking of archivering van externe bundles zonder de publieke hoofdroute te overschrijven.
