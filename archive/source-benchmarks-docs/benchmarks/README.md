# Benchmarks

Deze map bevat de eerste benchmarklaag voor de Shadow Seed Learning 4.5-repo.

## Doel

- benchmarks niet ad hoc maar via vaste lablogica inzetten
- SSL-onderdelen expliciet koppelen aan externe benchmarks
- reproduceerbare benchmarkvoorbereiding vastleggen
- voorkomen dat een niet-geverifieerde runner ten onrechte als echte benchmarkrun wordt gepresenteerd

## Startpunt

De benchmarklane begint standaard bij **AbsenceBench**.

Huidige status:

- benchmarkfocus: `AbsenceBench`
- executionstatus: `benchmarkvoorbereiding`
- execution-gap: `aanwezig`
