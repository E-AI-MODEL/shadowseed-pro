# Transfer Workspace

Deze map is bedoeld voor latere domeintransfer-evaluatie.

## Doel

Testen of seedkwaliteit overeind blijft buiten de bekende benchmarkdomeinen en buiten de huidige suite-stijl.

## Verwachte inputvorm

- extra domeinen
- cross-domain holdouts
- meerdere tekstgenres
- meerdere promptvormen

## Verwachte signalen

- acceptance rate per domein
- false-positive drift per domein
- probe utility per domein
- stabiliteit van atomiciteit buiten bekende suitevormen

## Grens

Transfer is pas zinvol als:

- open-set seedkwaliteit al bruikbaar is
- adversarial Gate-waarde al zichtbaar is

Deze map is dus bewust voorbereiding voor later, niet de eerstvolgende hoofdlaag.
