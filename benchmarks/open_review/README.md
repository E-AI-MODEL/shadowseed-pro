# Open review

This directory stores blind-review inputs and historical review rounds for open-set seed detection.

The active workflow keeps model output, mechanical prescreening, reviewer packets, answer keys, and aggregate summaries separate. A candidate is not treated as validated evidence merely because a model or automated prescreen accepted it.

Historical rounds are retained unchanged as regression fixtures. See `rounds/README_ACTIVE_FIXTURES.md` and `docs/migration/language-policy.md`.
